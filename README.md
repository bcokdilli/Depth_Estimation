## Usage/Examples

### CLI Usage
```bash
usage: cli.py [-h] input_image output_image

Depth estimation using ZoeDepth.

positional arguments:
  input_image Path to input image.
  output_image Path to output depth map.
```

### API Usage
```bash
http://127.0.0.1:8000/predict
```
